## Overview
This website is built from scratch using React, HTML5, and CSS3.

## Setup
1. Install the required packages:
```javascript
  npm install
```
2. Start a local server:
```javascript
  npm start 
```
3. Navigate to `http://localhost:4200/`.

### Linting
```javascript
  npm run lint
```

### Testing
```javascript
  npm run test
```
