## Overview
Personal portfolio built with [React](https://reactjs.org).

## Setup
1. Install the required packages:
```javascript
  npm install
```
2. Start a local server:
```javascript
  npm start 
```
3. Navigate to `http://localhost:8080/`.

### Linting
```javascript
  npm run lint
```
